Welcome to i22!

To the top left you will find a number of tabs (we call them perspectives) to carry out your experiment.

Scripts
Write and control scripts to automate tasks.
Setup
Run simple scans with 1D plotting. Useful for alignment and tests.
Ncd Detectors
Setup the detector system for Saxs/Waxs and configure the exposure timing and triggering.
Live Saxs
View the data as it comes of the Saxs detector.
Saxs Processing
Tools to get a more detailed view on the Saxs data.
Live Waxs
View the data as it comes of the Waxs detector.
